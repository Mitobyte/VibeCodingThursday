## üí° Tips for Designers

**When using these templates:**
1. Specify your brand colors and fonts
2. Mention target audience and mood
3. Reference design inspiration (URLs welcome)
4. Specify platform (web, mobile, desktop)
5. Note accessibility requirements

**Make prompts better by adding:**
- "Follow Apple/Google design guidelines"
- "Ensure WCAG AA compliance"
- "Use 8px grid system"
- "Include hover and active states"
- "Make it delightful to use"
- "Add smooth transitions"

**For design systems:**
- Start with design tokens
- Build atomic components first
- Document usage patterns
- Include accessibility notes
- Provide code + Figma files
